# Credit Card Fraud Detection using Machine Learning

## Overview

This project implements a Machine Learning-based Credit Card Fraud Detection system developed as part of the **Cryptography and Network Security** course.

The objective is to accurately classify fraudulent transactions using multiple machine learning algorithms while handling the severe class imbalance present in the dataset.

---

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Handling Imbalanced Dataset using SMOTE
- Feature Scaling
- Training Multiple Machine Learning Models
- Model Performance Comparison
- Fraud Prediction
- Model Explainability using SHAP

---
