# Credit Card Fraud Detection

## Overview
This project implements a fraud detection system using Machine Learning techniques to identify fraudulent transactions. The project was developed as part of the Cryptography and Network Security course.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Fraud Prediction

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Structure
├── FraudDectectionProject.ipynb
├── README.md
├── requirements.txt
└── .gitignore

## Installation

git clone <repository-url>

pip install -r requirements.txt

## Usage

Open the notebook:

jupyter notebook FraudDectectionProject.ipynb

## Results
The notebook demonstrates the complete workflow from preprocessing to fraud prediction.
